## 1. 기획 배경

전통적인 오프라인 교육 환경에서는 교사-학생 간 즉각적인 상호작용이 가능하나, 비대면 또는 하이브리드 학습 환경에서는 개인별 학습 진도 파악 및 맞춤형 피드백이 어려워졌습니다. 특히 학원, 재수학원, 과외, SSAFY, 기타 교육 기관에서는 학생 개개인의 학습 데이터를 실시간으로 수집·분석하여 즉각적인 학습 지원을 제공할 필요가 높아졌습니다.

본 프로젝트는 AIoT 센서를 통한 학습 상태 데이터와 실시간 통신 기능을 결합한 **CLASSE**으로, 학습자가 자율적으로 공부할 수 있는 "개인 공부방"과 관리자가 강의를 진행할 수 있는 "Class 공간"을 통해 학습 효과를 극대화하고자 합니다.

## 2. 사용 대상

- **학원** (어학, 수학, 과학 등)
- **재수학원** 및 대입 대비 학원
- **과외** (1:1 튜터링)
- **SSAFY** 및 기타 코딩 교육 기관
- **대학교/기업 교육 프로그램** 등 다양한 교육 기관

## 3. 플랫폼 구조 및 모드

### 3.1 개인 공부방 (Self-Study Room)

- **목적**: 학생이 혼자 학습할 때 AIoT 센서 데이터를 기반으로 집중도, 자세, 피로도를 실시간 모니터링
- **주요 기능**:
    - AIoT 센서를 통한 눈 깜빡임, 미동, 체온 등 생체 신호 수집
    - 집중도 하락 알림 및 학습 환경 개선 제안(예: 휴식, 스트레칭)
        - Redis→ 실시간 알림, 환경 개선 제안 → 알고리즘 필요
    - 개인 학습 진도 및 성과 대시보드
        - 추가: 성과 대시보드 + 집중도 개인 생체정보 통계 대시보드
    - 학습 리포트 자동 생성 및 이메일/알림 발송
        - SMTP or Redis

### 3.2 Class 공간 (관리자 감독형)

- **목적**: 강사(관리자)가 다수의 학생을 온라인으로 관리하면서 강의, 화면 공유, 채팅 등을 수행
- **주요 기능**:
    - 실시간 강의 스트리밍(화상·음성) 및 (화면 공유), 채팅
    - 학생별 AIoT 집중도 현황 모니터링 뷰
        - 출석 체크 및 참여도 기록(선택)

- 학생 개인 맞춤형 학습 지원으로 학습 효과 향상
- 관리자는 Class 공간에서 실시간 학생 상태 모니터링 및 피드백 제공
- AIoT 기반 데이터로 학습 환경 최적화
