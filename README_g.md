## Note
### `ColorizedImage`
Build and install the Qt Design Studio QML modules (https://github.com/qt-labs/qtquickdesigner-components) to import `QtQuickUltralite.Extras` for `ColorizedImage`.


### Force to Enable Virtual Keyboard 
```
QT_QPA_PLATFORM=xcb QT_IM_MODULE=qtvirtualkeyboard python3 main.py
```
